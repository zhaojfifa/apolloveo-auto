创建矩阵脚本任务

这里只填写创建任务所需的产线输入。脚本变体、槽位细化和交付发布在任务创建后的工作台与交付页完成。

仅填写不透明引用：content://、task://、asset://、ref:// 之一，或令牌 ID。不接受外部网页 URL（https:// / http://）和对象存储桶（s3:// / gs://）；这些不是产品内部可解析的不透明引用。不要把脚本正文粘贴进来——脚本正文由后续工作台阶段处理。

推荐方式（Plan E Option F2 已上线）：点击「铸造句柄」按钮，由产品分配一个全新的不透明 content://matrix-script/source/<product-minted-token> 句柄。产品拥有 <token> 的分配与唯一性。

过渡兼容：原 §8.F / §8.H 操作员自定 <token> 形式仍然接受（例如 content://matrix-script/source/op-token-001）；既有句柄无需迁移，因为它们与铸造句柄同属 content:// 不透明方案集。task:// / asset:// / ref:// 形式同样接受。
