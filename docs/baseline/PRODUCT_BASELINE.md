# Product Baseline

## This Layer Is For

本文件冻结当前项目的产品与业务现实，不描述未来平台理想图。

## Current Reality

- ApolloVeo 当前仍是以成片交付为目标的 AI 内容生产工厂
- 当前工程主线是 Hot Follow sample stabilization 与 controlled continuation，而不是多产线扩张
- Task → Workbench → Publish 的业务骨架已成立
- Hot Follow 已通过真实业务使用验证以下结果：
  - parse success 展示正确
  - translated subtitle editing 维持 SRT-first
  - final compose duration 跟随 video length
  - final output 可在真实业务流中使用
- Hot Follow 现已成为当前唯一完成 Skills MVP v0 closure 的业务样本：
  - advisory skill 已存在
  - advisory 仅为 read-only、non-blocking operator guidance
  - advisory 不拥有 compose / publish / status truth

## Partial Reality

- 业务主链可用，不代表整套 runtime / skills 已 fully generalized
- workbench/detail/publish 的 full line-aware 统一仍未完成
- 当前业务验证重点仍在 Hot Follow，而不是跨线复用验证
- 当前产品重点仍是稳定化与受控扩展，不是 multi-line proliferation

## Postponed Work

- Skills generalized implementation / platformization
- second production line rollout
- provider expansion as a primary objective
- OpenClaw 作为当前执行主题
