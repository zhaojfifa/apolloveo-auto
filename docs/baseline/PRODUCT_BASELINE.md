# Product Baseline

## This Layer Is For

本文件冻结当前项目的产品与业务现实，不描述未来平台理想图。

## Current Reality

- ApolloVeo 当前仍是以成片交付为目标的 AI 内容生产工厂
- 当前工程主线是 Hot Follow cleanup，而不是多产线扩张
- Task → Workbench → Publish 的业务骨架已成立
- Hot Follow 已通过真实业务使用验证以下结果：
  - parse success 展示正确
  - translated subtitle editing 维持 SRT-first
  - final compose duration 跟随 video length
  - final output 可在真实业务流中使用

## Partial Reality

- 业务主链可用，不代表整套 runtime 已 fully generalized
- workbench/detail/publish 的 full line-aware 统一仍未完成
- 当前业务验证重点仍在 Hot Follow，而不是跨线复用验证

## Postponed Work

- Skills MVP implementation
- second production line rollout
- provider expansion as a primary objective
- OpenClaw 作为当前执行主题
